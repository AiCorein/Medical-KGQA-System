# Medical-KGQA-System
### 1、简介

​	简单的药品知识图谱问答系统。

​	知识图谱服务构建使用 Apache Jena。前端构建使用 Vue3 + Vite + Element plus。后端服务由 python flask 提供，使用 ltp 进行问句解析后，发送给知识图谱服务进行查询，随后将数据格式化后返回给前端，即实现了问答功能。
